# CodeAlpha_PhishingAwarenessTraining


This repository contains a **Phishing Awareness Training presentation** created as part of the **CodeAlpha Internship Program**.
The presentation is designed to **educate users about phishing attacks, social engineering tactics, and best practices** to stay safe online.

---

## 📑 Contents

* `Phishing_Awareness_Training.pptx` → Training presentation (PowerPoint)
* Covers the following key areas:

  * ✅ What is Phishing?
  * ✅ How to recognize phishing emails & fake websites
  * ✅ Social engineering tactics used by attackers
  * ✅ Best practices & security tips
  * ✅ Real-world phishing examples
  * ✅ Interactive quizzes for engagement

---

## 🚀 Features

* Easy-to-understand **slides with visuals and examples**
* Focus on **practical tips** to avoid phishing attempts
* Includes **real-world attack scenarios**
* Engaging **quiz questions** for learners

## 🎯 Learning Outcomes

By the end of this training, learners will be able to:

* Understand the **basics of phishing** and how attackers operate.
